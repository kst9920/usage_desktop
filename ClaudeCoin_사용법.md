# ClaudePet 🐑 — 데스크탑을 걸어다니는 꼬마 양 Claude 사용량 위젯

꼬마 양이 작업표시줄 위를 걸어다니며 Claude 사용량을 보여줍니다.
창을 만나면 꿈틀꿈틀 타고 올라가거나 방향을 바꾸고, 창 끝에서는 아래로 떨어집니다.

> `claude_pet.py`가 최신 버전입니다. (`claude_coin.py`는 구버전 — 지워도 됩니다)

## 표시 내용
- 배: 현재 세션(5시간) 사용량 % + 리셋까지 남은 시간 (예: `4h15m`)
- 목도리 색: 파랑(<50%) → 주황(<80%) → 빨강(80%+)
- **클릭**: 주간 한도 패널 — 모든 모델/Opus/Sonnet 사용량 %와 초기화까지 남은 일수·시간

## 조작
| 동작 | 기능 |
|---|---|
| 좌클릭 | 주간 한도 패널 열기/닫기 |
| **더블클릭** | 캐릭터 랜덤 교체 (고양이/강아지/펭귄 …) — 패널은 안 뜸 |
| 드래그 | 집어서 이동 (놓으면 떨어짐) |
| 우클릭 | 새로고침 / 캐릭터 바꾸기 / 걷기 켜기·끄기 / 종료 |

## 실행 방법
1. Python 3.9+ 설치
2. **Pillow 설치 (부드러운 그래픽에 필요)**:
   ```
   pip install pillow
   ```
   없으면 실행은 되지만 거친 동전 모양으로 나옵니다.
3. **Claude Code에 로그인되어 있어야 합니다** (`claude` 실행 후 로그인)
4. 콘솔 창 없이 실행:
   ```
   pythonw claude_pet.py
   ```

## 커스텀 캐릭터
`skin` 폴더 안에 **캐릭터별 하위폴더**를 만들고 PNG를 넣으면 캐릭터 세트가 됩니다.
시작할 때 랜덤으로 하나가 선택되고, 더블클릭할 때마다 다른 캐릭터로 바뀝니다.
(기본 양은 skin 폴더가 비어 있을 때만 나옵니다)

```
skin/cat/     walk1.png walk2.png climb1.png climb2.png fall.png idle.png
skin/dog/     (같은 구성)
skin/penguin/ (같은 구성)
skin/새캐릭터/ ← 폴더만 추가하면 자동 인식
```

- 이미지 규격: 정사각형(권장 512×512), 투명 배경, **오른쪽을 보는 그림**
- 일부 포즈만 넣어도 됨 (없는 포즈는 그 세트의 다른 그림으로 대체)
- 사용량 %와 시간 텍스트는 그림 중앙에 자동으로 얹힘 → 중앙은 밝고 단순하게
- 현재 cat / dog / penguin 3종 설치됨 (원본 시트 cat.png, dog.png, pagnguin.png는 지워도 됨)

## 시작 프로그램 등록 (선택)
`Win+R` → `shell:startup` → 바로가기 생성, 대상:
```
pythonw.exe "C:\workspace\dev\bomnal\usage_desktop\claude_pet.py"
```

## 배포용 exe 만들기
폴더의 **`build.bat`를 더블클릭**하면 끝. `dist\ClaudePet.exe`가 생성됩니다.

- skin 폴더(고양이/강아지/펭귄)가 exe 안에 포함됨 → **파일 하나만 전달**하면 됨
- 받는 사람은 Python 설치 불필요
- 단, 받는 사람 PC에 **Claude Code 로그인**이 되어 있어야 사용량 표시 (본인 계정 사용량이 보임)
- exe 옆에 skin 폴더를 따로 두면 내장 스킨 대신 그 폴더를 사용 (캐릭터 추가 가능)

## 참고
- 사용량은 3분마다 갱신 (429 발생 시 자동으로 간격 증가)
- 토큰 만료 시 자동 갱신 시도, 실패하면 `!` 표시 → Claude Code를 한 번 실행하면 해결
- 비공식(내부) API 사용 — Anthropic이 언제든 변경할 수 있음
